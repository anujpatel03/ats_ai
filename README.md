# AI-Powered ATS Resume Advisor
- AI-powered resume analysis tool designed to evaluate and match resumes against job descriptions.
- Using **gemini-1.5-flash AI models**, it provides detailed feedback on the strengths, weaknesses, and overall fit of a resume, suggests improvements, and calculates a percentage match, helping users optimize their resumes for better job prospects.
